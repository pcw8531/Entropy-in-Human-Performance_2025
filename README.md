# Data and Analysis for "Strategic Entropy in Human Performance"
This repository contains the data and simulation code supporting the manuscript by Park (2025).

## Contents
- `circadian_analysis.ipynb`: Analysis of circadian motor coordination data
- `expertise_analysis.ipynb`: Movement-perception expertise analysis
- `/data`: Core datasets used in the study
- `/simulations`: Simulation code for theoretical models

## Requirements
- Python 3.x
- NumPy, SciPy, Matplotlib, Pandas
- Jupyter Notebook
